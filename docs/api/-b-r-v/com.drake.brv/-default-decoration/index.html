最强大的分割线工具

  1. 分隔图片

  2. 分隔颜色

  3. 分隔间距

  4. 回调函数判断间隔

  5. 首尾是否显示分隔线, 可以展 示表格效果

  6. 类型池来指定是否显示分割线

  7. 支持全部的LayoutManager, 竖向/横向/网格分割线

  8. 优于其他框架, 完美支持均布网格分隔物

  9. 支持分组条目的分割线
